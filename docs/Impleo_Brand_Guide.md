
# Impleo Design System v2

## Brand Identity
Change the Extention Name To Impleo
Impleo is an AI-powered autofill assistant that feels:
- Adaptive
- Fast
- Trustworthy
- Playful
- Premium
- Professional

The mascot is a smart chameleon that adapts to any form or website.

## Typography

Primary Font:
- Geist

Fallback:
- Inter

Sizes:
- App Title: 18px
- Card Title: 14px
- Body Text: 13px
- Metadata: 12px
- Caption: 11px

Use compact typography similar to:
- MongoDB Atlas
- Linear
- Raycast

---

## Component Rules

Border Radius:
- Cards: 12px
- Buttons: 10px
- Inputs: 10px

Animations:
- 150ms to 250ms transitions
- Smooth hover states
- No flashy motion

Shadows:
- Very soft only
- Premium SaaS appearance

---

## UI Personality

The UI should feel like:

MongoDB Atlas + Linear + Arc Browser + Raycast

Avoid:
- Bootstrap appearance
- Large fonts
- Excessive gradients
- Corporate enterprise styling

Prioritize:
- Small typography
- Compact layouts
- Premium whitespace
- Trust and clarity

---

## Extension Workflow

Form detected ->
AI extracts fields ->
User reviews suggestions ->
Approve/Edit/Skip ->
Fill approved fields

The user must always feel in control.
